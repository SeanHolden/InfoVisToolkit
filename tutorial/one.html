Tutorial one:

Intoduction and Creation of a Basic Chart

See completed code

1: Setting up the file structure show

Before we begin writing any code, we need to ensure our file structure is correctly set up.

Firstly create a new directory. We can call it: ChartApplication.

Navigate into this directory and create three more directories inside. Name them css, js and img, respectively. You may have guessed already, but these directories will be used to hold your stylesheets, javascript and images.

Next we want to create our main HTML file. We will call it index.html. This should be placed alongside your other three directories.

So far your file structure should look something like this:

                      ChartApplication/
                             |
                             |----css/
                             |----img/
                             |----js/
                             |----index.html

Inside your css directory, create a new file. Call it main.css. This will be our main stylesheet we can use to style our application.

Next, create a new file inside your js directory. Sticking with the same style as with css, we will call this one main.js. Alongside this new file, create a new directory called libraries. This is where we will store all third party libraries that will make our job easier.

Next up we need to download our first library, jQuery. At the time of writing this tutorial, the latest stable version is 1.11.0 and this is the version that was used to create the main application. You can get it directly from here or simply go to jquery.com/download and click on your preferred version. Save this as a new file inside your newly created libraries directory. Call it jquery-1.11.0.js (or if you are using a newer version, ensure to name it accordingly).

You structure should look like the following:

                      ChartApplication/
                             |
                             |----css/
                             |      |
                             |      |----main.css
                             |
                             |----img/
                             |
                             |----js/
                             |      |----main.js
                             |      |----libraries/
                             |                   |----jquery-1.11.0.js
                             |
                             |----index.html

2: Preparing the HTML foundations show

Lets get started by adding some HTML code into index.html.

The following code should be written (or copy and pasted) directly into index.html.

<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8">
    <title>Information Visualization Toolkit</title>
    <meta name="viewport" content="width=device-width">
    <link rel="stylesheet" href="css/main.css">
  </head>
  <body>
    <!-- Content to go here -->
  </body>
</html>

Line 01 is declaring this to be an HTML5 document. Inside the head tags we have the some meta tags. The meta tag on line 04 is simply stating that we want to use utf-8 character set, and line 06 is to ensure a nicer mobile experience by setting the width to the width of the device.
We have also set the page title and included a link to our main.css file which means that when we add some styles they will take effect straight away.

At this point we can add our JavaScript files main.js and our library jquery-1.11.0.js.
Add the following script elements inside the body tags: 

  ...
  <body>
    <!-- Content to go here -->
    <script type="text/javascript" src="js/libraries/jquery-1.11.0.min.js">
    </script>
    <script type="text/javascript" src="js/main.js"></script>
  </body>
  ...

We have just linked our JavaScript files to our HTML document. These scripts are placed at the bottom of the document as the page is read by the browser from top to bottom. Placing JavaScript files near the top can result in slower page loadig times.

At this point we are almost ready to start creating a chart. But first we need to gather a few more libraries. This time, we need libraries that are made for plotting charts.

3: Acquiring the necessary libraries show

We already have the jQuery library, now it is time to grab a plotting library. Creating a chart from scratch would be a tedious and long process. There are many libraries out there that already do this hard work for us. The one used for this application will be FlotCharts which is available at http://www.flotcharts.org/.

Navigate to http://www.flotcharts.org/ and download the latest version. It will come in a zip file.

Unzip this file. You will notice that there are many files inside, but there is no need to be overwhelmed. We only actually need two of these files at this point.

Find flot/jquery.flot.min.js and flot/jquery.flot.stack.min.js. The former of these files is the core library and the latter is an add-on that adds stacking functionality to a created chart.

Once these files are located, copy them into your js/libraries directory.

Now we can link to them from our index.html file as we did with our other JavaScript files. Place the script tags between your jQuery library and main.js. The bottom of your body tags should look like the following: 

  ...
  <body>
    <!-- Content to go here -->
    <script type="text/javascript" src="js/libraries/jquery-1.11.0.min.js">
    </script>
    <script type="text/javascript" src="js/libraries/jquery.flot.min.js">
    </script>
    <script type="text/javascript" src="js/libraries/jquery.flot.stack.min.js">
    </script>
    <script type="text/javascript" src="js/main.js"></script>
  </body>
  ...

Now we are pretty much set up. We have our required libraries, we have linked them to our HTML document as well as our own JavaScript file main.js and our stylesheet main.css. The next section will go into plotting your first chart.

4: Plotting your first chart show

First, go back into your index.html file and add the following line into your body element.
Note: Ensure that this line is placed above the script elements. The script elements must always be at the bottom of the body. 

  ...
  <body>
    <div id="placeholder"></div>
    ...
  </body>
  ...

The div element that we have just inserted will be the container for our chart. The whole chart will be generated via JavaScript and will be placed inside this div element. We have given it the id of "placeholder". This can be renamed to anything, but this id will be used in our JavaScript code to reference this element, so ensure it is consistent throughout your code.

At this point, we need to set the height and width of our chart area. We can do this in our main.css file. Open this stylesheet and insert the following code: 

  #placeholder{
    width:600px;
    height:300px;
    border: 2px solid black;
  }

Notice that as well as height and width, we have set a border. This is simply so that we can see our chart area before our chart is even placed inside it.

If you open index.html at this point in your web browser, you should see a black rectangular box with a width of 600 pixels and a height of 300 pixels. It should look like this:

Now for the fun part! Open up main.js. We are ready to write some JavaScript and get our first chart displaying.

First of all, we need to write some code that will display the chart as soon as our HTML document has loaded in the browser. Fortunately, jQuery makes this easy for us. Simply type the following couple of lines.

  $(function(){
    // code will go here
  });

This is essentially a jQuery helper function that says "any code inside these brackets will run whent he document loads".

Now lets add a line of code to set us up to display our chart. In place of the comment that says: "// code will go here", insert the following line:

  $(function(){
    displayChart(); // <-- insert this line
  });

This will not work just yet, as the browser still does not know what displayChart() means. Lets create this function so that it can be ran as expected.

Edit the code to become the following:

  $(function(){
    displayChart();
  });
   
  function displayChart(){
    // x and y coordinates:
    var values = [ [0, 3], [2, 8], [4, 5], [6, 13], [8,0], [10,5] ];
    // flotcharts function:
    $.plot("#placeholder", [ values ]);
  }

Now hit refresh in your browser!

All going well, you should see a line chart displayed with the X and Y values taken from the "data" variable in the code snippet above.

You might want to remove the border in your main.css at this stage, to make it look a little cleaner.

Your chart should look something like this:

Congratulations! You have just created your first chart in HTML5!

5: Change Chart Type show

Now we have a line chart. But what if we want to display a column chart? or a points chart? or multiple chart types at once?

Going back to our JavaScript code in main.js, we need to make an edit to our $.plot line inside the displayChart() function.

By default, the chart will display a line chart. In order to change this, we need to add some options into our code.

  ...
 
function displayChart(){
  // x and y coordinates:
  var values = [ [0, 3], [2, 8], [4, 5], [6, 13], [8,0], [10,5] ];
    $.plot("#placeholder", [{
      // adding extra options:
      data: values,
      bars: { show: true }
    }]);
}

What have we just done here?
Well, the $.plot function takes an array as its second argument (the first being the id of the HTML element). Initially we passed in the value variable which already contained an array of arrays. As we did not specify anything other than this, the library used its default values, namely choosing to display a line chart.

If, instead of entering the value variable as an argument, we enter it as part of an object, then we can override these default values. This may sound confusing, but if you study the code snippet above, you can see that two "options" have been set here. One being the data option, which we set as our value variable. Next is the bars option, which accepts multiple options itself. One of which is show, which in this case we have set to true.

Reload the browser page again. You should see that the chart is now a column chart!

To expand on this further, how about we add some extra data to the chart, in the form of a line chart with points.

Edit your displayChart() function to look like the following:

  ...
 
function displayChart(){
    var values1 = [ [0, 3], [2, 8], [4, 5], [6, 13], [8,0], [10,5] ];
    var values2 = [ [0, 7], [3, 4], [5, 1], [7, 2], [9,5], [11,10] ];

    $.plot("#placeholder", [{
      data: values1,
      bars: { show: true }
    },
    {
      data: values2,
      lines: { show: true },
      points: { show: true }
    }]);
}

Here, you may notice that we now have two values variables, named values1 and values2. Inside the $.plot function, we now have an array containing two objects. The first object is to display the column chart as before. The second object, having lines and points values set to true, will display a line chart with points.
